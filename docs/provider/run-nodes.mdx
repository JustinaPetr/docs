# Node Configuration

The first step in your RPC configuration setup is runnin a node for the chain you would like to support.
Each chain has different server requrements and configuration steps. It is always best to use the official documentation guides for setting up the nodes. 
Below is a list of supported chains and links to their official configuration guides:
